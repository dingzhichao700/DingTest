UI快速构建工具: UIFastBuilder 

【v1.0.1】：
功能：
1、组件的手动布局、参数调整；
2、组件布局信息的导入导出（xml文件形式）；
3、代码直接输出（AS3代码形式）；

使用说明：
1、鼠标可从右下方组件库中拖放组件到舞台上；
2、选中单个组件后，鼠标滚轮调整宽度，按住ctrl调整高度，按住shift每次调整10像素；
3、通过“上移”“下移”或“PageUp””PageDown”调整组件在舞台上的层级；
4、按下ctrl后可用鼠标框选多个组件；
5、在左侧属性输入框中输入信息，按回车使对应属性生效；

目前包含组件：
TextField 文本
Bitmap 位图
Button 按钮
Tabbar 切换标签
panelBg 窗体背景（九宫格图）
